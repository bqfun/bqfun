加工済みオープンデータ

BQ FUN でいくつかのデータを加工して、Google Cloud Analytics Hub で公開しています。 Google Cloud 利用者は、自身の BigQuery データセットにリンクして、自動更新されるデータを利用できます。

PODB on BigQuery もご確認ください。

データ一覧​

データ名データセット名更新頻度Analytics Hub
法人情報（経済産業省）gbizinfo毎日US, asia-northeast1
法人情報（国税庁）houjinbangou毎月US, asia-northeast1
国民の祝日holidays毎日US, asia-northeast1
住所、所在地（デジタル庁）base_registry_address毎日US, asia-northeast1
SSDSE（教育用標準データセット）ssdse毎日US, asia-northeast1
